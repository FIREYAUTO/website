Storage Library

Classes

The Storage class is a base class. This class holds all the properties for the other storage classes

new Storage(Name, StorageObject)

The LocalStorage class extends the Storage class. This class represents a key in window.localStorage

new LocalStorage(Name)

The SessionStorage class extends the Storage class. This class represents a key in window.sessionStorage

new SessionStorage(Name)

Methods

GetItem

Returns the given item stored inside the Storage object

Storage.GetItem(Name)

Example:

Storage.GetItem("Test")

GetItems

Returns all items stored inside the Storage object

Storage.GetItems()

SetItem

Stores the given Name as Value in the Storage object

Storage.SetItem(Name, Value)

Example:

Storage.SetItem("Test",true)

RemoveItem

Removes the given Name from the Storage object

Storage.RemoveItem(Name)

Example:

Storage.RemoveItem("Test")

Check

Calls the Callback function with Name in the Storage object, and returns if the item matches the condition

Storage.Check(Name, Callback)

Example:

Storage.Check("Test",Item=>Item===true)

__SO

The internal storage object

Storage.__SO

__NAME

The internal name of the storage

Storage.__NAME

__GET_STORAGE

An internal function with has the same functionality as Storage.GetItems()

Storage.__GET_STORAGE()

__SET_STORAGE

An internal function which sets the storage object in its internal storage

Storage.__SET_STORAGE(Object)

Library Example

const MyStorage = new LocalStorage("MyStorage");
MyStorage.SetItem("Test",true);
console.log(MyStorage.GetItem("Test"));
console.log(MyStorage.Check("Test",Item=>Item===true));
MyStorage.RemoveItem("Test");
console.log(MyStorage.GetItem("Test"));
